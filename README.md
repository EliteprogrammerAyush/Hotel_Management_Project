# Hotel Management and Food Ordering System

## Project Description
This is a simple **C++ console-based project** to manage hotel rooms and food orders.  
You can:
- Allocate rooms (1BHK, 2BHK, 3BHK)
- Take orders for food items like Pasta, Noodles, Burger, Paneer, and Shakes
- Track sales and total collection

---

## Features
- Enter available quantity of rooms and food items
- Check availability before allocating rooms or taking orders
- Calculate total sales for each item

---

## Prices
- **Rooms:** 1BHK/2BHK/3BHK – ₹1200 each  
- **Food Items:** Pasta – ₹200, Noodles – ₹90, Burger – ₹80, Shake – ₹120, Paneer – ₹120

---

## How to Run
1. Copy the code into a C++ IDE or compiler  
2. Compile and run  
3. Follow the menu to allocate rooms or order food
